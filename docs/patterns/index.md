# Pattern catalogue

vstack ships 34 patterns across three modules:

- **Module 1 — Individual agent** (12 patterns) — Lewin attribution, Goleman EI, Johari, DANVA emotion reader, cognitive reappraisal, Yerkes-Dodson workload, HEXACO personality, Grant strengths-as-weaknesses, motivation traps, SDT intrinsic reward, McGregor orchestrator mode, Vroom expectancy.
- **Module 2 — Multi-agent team** (18 patterns) — GRPI working agreement, process gain/loss, social loafing, superflocks, Lencioni five dysfunctions, Trust Triangle, McAllister trust, Edmondson psychological safety, Glaser conversational intelligence, Stone-Heen feedback triggers, Plus/Delta feedback, SMART goal generator, group decision protocols, debate pathology, bias stack, devil's advocate separator, Thomas-Kilmann conflict styles, After-Action Review (foundational).
- **Module 3 — Organizational** (4 patterns) — Schein iceberg culture, Robbins-Judge 7-characteristic culture, org-structure matrix, span-of-control.

The full index with citations lives in [`PATTERNS.md`](https://github.com/valani9/vstack/blob/main/PATTERNS.md); academic anchors per pattern are in each pattern's `CITATIONS.md` (also exposed at `vstack://patterns/<name>/citations` via MCP).

## How to pick one

Don't memorize all 34. Either:

- Use the [composition runbook](../concepts/composition.md) — one of the five canonical chains usually covers what you need.
- Invoke `/vstack-pick-pattern` in your MCP client — it interviews you in two questions and recommends 1-3 patterns.
- Browse `vstack://patterns/index` (MCP resource) for the live catalogue with summaries.

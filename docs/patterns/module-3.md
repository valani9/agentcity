# Module 3 — Organizational patterns

4 patterns operating at organization / structural / culture scale.

1. **#31 Schein iceberg** — `vstack_schein_culture` — three-layer culture model (artifacts / espoused values / underlying assumptions). The deepest culture diagnostic in vstack.
2. **#32 Robbins-Judge** — `vstack_robbins_culture` — seven-characteristic culture profile + type label.
3. **#33 Org-Structure Matrix** — `vstack_org_structure` — Galbraith-Mintzberg six-dimension structural-fit diagnostic.
4. **#34 Span-of-Control** — `vstack_span_of_control` — deterministic graph metrics (max_span, mean_span, centralization_index, hierarchy_depth, span_gini, decision_bottleneck). The math is gated to Python; the LLM only paraphrases.

The deterministic Span-of-Control numbers + pre-shipped canonical baselines (see [Calibration baselines](../concepts/baselines.md)) make this module the most ready for monitoring use cases.

See [PATTERNS.md](https://github.com/valani9/vstack/blob/main/PATTERNS.md) at the repo root for full per-pattern detail including citations.
